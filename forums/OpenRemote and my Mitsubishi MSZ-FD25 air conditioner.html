Here is my first approach to control Mitsubishi MSZ-FD25 air conditioner from OpenRemote system. OR-controller talks with http2serial server (my modification of Jon Berg's Simple Java web server, added with some serial communication and status handling) which talks with KM08ARSUSB USB IR module. I tried to keep the panel as simple as possible so I just added functions that would be needed when using aircon in the winter.
