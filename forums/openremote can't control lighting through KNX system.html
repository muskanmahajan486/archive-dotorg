There's a bug in the Windows 7 networking code, so gateway discovery won't work there without a patch (http://openremote.org/display/forums/KNX+conn.+manager+fails+to+start?focusedCommentId=9273352#comment-9273352), so try with Windows XP only unless you've patched your version for Windows 7.

In your /logs/controller.log you can find more details on what goes on with the KNX discovery, see for example : http://openremote.org/display/forums/KNX+conn.+manager+fails+to+start?focusedCommentId=8749102#comment-8749102
