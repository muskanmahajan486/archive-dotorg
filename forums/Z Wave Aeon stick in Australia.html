Comments:

I don't live in Australia, so this answer might be wrong but I would not expect the different frequency to cause issues.

Z-wave uses different frequencies in US/EU (and AUS apparently) due to regulation. So you need to make sure ALL your Z-wave devices are in the same frequency band (so buying them locally when possible is a safe bet). But as far as OR goes, we interface with the Z-stick via its serial protocol (through USB port) so we have no knowledge of the physical layer of the network – the frequencies used, etc. So therefore I would expect it to work normally.

But unless someone else from Australia confirms this, you may have to be the first one to give it a try. If so, please do report back here (success or no success) for the next Australian visitor to benefit from that knowledge
