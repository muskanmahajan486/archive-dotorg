WuKong project will give away 5 sets of WuDevice with Z-Wave module for the OSS community. The WuDevice is specially designed for the Z-Wave and XBee. Currently, the Z-Wave is officially supported by the WuKong project. The WuDevice does not only support the WuKong middleware, but also Arduino IDE. We can use to as bridge to support Z-Wave in either openremote or openzwave. Since there is AtMega2560 CPU in the WuDevice, it can provide much more flexibility for the Z-Wave community to create Z-Wave/Xbee applications.

The WuDevice has an enclosure, which is specially designed for it. For more information, please check the WuKong website(https://sites.google.com/site/arduinozwave/). The HC-20 is currently available at eBay(http://www.ebay.com/sch/i.html?_trksid=p5197.m570.l1313&_nkw=arduino+zwave&_sacat=0&_from=R40).
