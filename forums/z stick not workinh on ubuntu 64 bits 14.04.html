Hi,
I dont know anything about Linux but to me it seems like 32-bit vs 64-bit issue. Same thing applies to running OpenRemote on Windows, where seems to be no support running under 64-bit architecture since serial drivers in openremote is 32-bit. The solution for my Windows installation was to run it under 32-bit OS and Java or download a 64-bit serial driver and replace the one in Openremote. I downloaded it from http://fizzed.com/oss/rxtx-for-java and there seems to be some Linux drivers there as well.

As I mensioned, dont know anything about Linux but maybe that can help you in the right direction.

/Nils
