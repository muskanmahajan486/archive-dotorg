Pardon, mi espanol no es bueno. Si, el controlador trabaja in Windows 8.

I just tested using the instructions for Windows XP at http://www.openremote.org/display/docs/OpenRemote+2.0+How+To+-+Controller+Installation

Basically
1. Install Java
2. Set JAVA_HOME environmental variable
3. Download and unzip OpenRemote
4. Using the cmd terminal navigate to the Openremote\bin location and type openremote.sh run
5. On your browser, type http://localhost:8080/controller to get the login screen
6. You might want OR script to start automatically on reboot. You can google how to do this.
