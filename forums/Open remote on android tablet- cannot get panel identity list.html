This is likely a network issue:

1) Make sure your tablet is connecting over WiFi, not 3G

2) Check that from your tablet's web browser you can connect to the controller. Enter the URL [WinXP Controller IP Address]:8080/controller

Note that if your WinXP machine has multiple network interfaces, it may have multiple IP addresses
