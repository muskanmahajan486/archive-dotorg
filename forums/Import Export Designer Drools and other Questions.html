Comments:

You can deploy a decision table as CSV file. At the moment there's no way to do this through designer, so you will need to drop the CSV file into the controller's rules directory by yourself.

There's a example of CSV in the test suite (https://openremote.svn.sourceforge.net/svnroot/openremote/tags/project/Controller/Controller_2_0_1/test/org/openremote/controller/fixture/statuscache/rules/dtable/rules/DTableTest.csv) but it does lack formatting so a tad difficult to read. Or check the Drools documentation on creating your own decision tables. I haven't yet had time to build more meaningful examples around decision tables.

Import/Export is being actively worked on by Eric.
