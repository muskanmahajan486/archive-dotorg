It remounts the file system as read/write. Default is read only. However, apt-get update should be safe, although it can nuke something if you are not prudent (don't fix it if it doesn't break!). After you are done then:

remountro

It is not dangerous per-se, my eBox is already few years in r/w mode up and running. As long as you know what are you doing you'll be OK.
