You'll probably need the RXTX serial communication library compiled for the ARM architecture.

There's a solution described in the comments section of the OpenRemote 2.0 How To - Install Controller on Raspberry Pi page:

Solved by:

sudo apt-get install librxtx-java

Link or copy /usr/lib/jni/librxtxSerial.so to OR location /webapps/controller/WEB-INF/lib/native/librxtxSerial.so
