Hi,

Welcome to OpenRemote first of all.

To try and answer your questions: -

1). There are some performance issues with the Designer which become more noticeable with larger projects; Juha has done some work to try and improve things so I don't know the current state of these designer performance issues. All I can suggest is to try using as new a browser as possible (Google Chrome is the best option if you are willing/able to try this). Also what is your internet connection speed like? If it is very slow then the RPC calls to the OpenRemote servers might be timing out before a response is received.

2). When you arrive home does your iPhone connect to the wifi and then you try and load OpenRemote or are you loading OpenRemote and connecting to your home via your Mobile Data connection?

Sounds like you have some network issues that you need to resolve; when you say the Controller is sometimes not available, are you sure the controller is not working or could it be the device you are running OpenRemote console on just doesn't have a network connection to the server?
