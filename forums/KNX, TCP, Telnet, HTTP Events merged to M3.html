Forums : KNX, TCP, Telnet, HTTP Events merged to M3
This page last changed on Jun 18, 2009 by juha.

Merged two feature branches into trunk (on its way to become Milestone 3). These include support for KNX, TCP, Telnet and HTTP Get events which can act on an input from the iPhone user interface.

They can be configured directly using XML, just like other event types. The plan is to support the XML generation from the UI composer interface with the upcoming M3 release.
