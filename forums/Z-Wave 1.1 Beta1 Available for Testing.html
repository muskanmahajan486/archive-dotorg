Forums : Z-Wave 1.1 Beta1 Available for Testing
This page last changed on Aug 09, 2013 by juha.

This is a minor feature update to the free Z-wave implementation.

You can download it from http://download.openremote.org/free/zwave/1.1/beta1

Changes:

- Adds support for distinct DIM level commands such as 'DIM 25'
- Supports state sensor binding to Z-Wave binary sensors

We'd particularly like feedback on using the distinct dim levels, so please leave a response on the forums.
