I don't believe there has been any changes and think Serial was never in the "main" designer.

There are 3 different versions of the designer available: main stable one, staging and preview.
The later 2 are for testing some beta features before including in the main one.
I suppose you were using one of those before.

Please note that they are all connected to the same database so they share the same data.
This means that if you use a new feature in staging/preview and go back the main one, you may see errors, empty GUI design, ... and can potentially corrupt all your data. So it's always better to stick to the same version of the designer to avoid those issues.
