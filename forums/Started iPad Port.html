Apparently porting the first version was not a huge undertaking - it's a pretty straight-forward port but does not take advantage of the new UI components iPad has that are not included with iPhone SDK.

It has not been tested on actual hardware yet, only the simulator, so that will take a couple of weeks first and then the approval process with Apple which is going to be another couple of weeks.

It will all be released during this summer time. In the meantime, if you have the tools, you can pull the code from subversion.
