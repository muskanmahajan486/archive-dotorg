Hi,

I'm trying to create sensor in order to get status of my Yamaha RX-A810 AV receiver.

Thus I have created this command :

<command id="203" protocol="http">
<property name="workload" value="<YAMAHA_AV cmd="GET"><Main_Zone><Basic_Status>GetParam</Basic_Status></Main_Zone></YAMAHA_AV>" />
<property name="pollingInterval" value="5s" />
<property name="method" value="POST" />
<property name="url" value="http://192.168.0.12/YamahaRemoteControl/ctrl" />
<property name="xpath" value="/YAMAHA_AV[1]/Main_Zone[1]/Basic_Status[1]/Input[1]/Input_Sel[1]" />
<property name="name" value="GetParam_Source" />
</command>

It's working perfectly when using last release of controller instead of 2.0.1.0008

OpenRemote-Controller-2.1.0_SNAPSHOT-2013-04-09

POST was sent via a GET in buggy 2.0.1.008 release of controller.
