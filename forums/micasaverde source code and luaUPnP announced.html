So the micasaverdeguys just released their source code at http://wiki.micasaverde.com/index.php/Source_Code.  They also announced http://wiki.micasaverde.com/index.php/LuaUPnP for doing IR, A/V and serial control.  Interesting stuff.

For those who have not followed micasaverde, its bascially the core messaging and device control parts of the Pluto/LinuxMCE project that have been simplified and focused on basic home automation control for use on a emeded device.  Their hardware is bassically a linksys or buffalo style router (think dd-wrt compatible) retooled and branded for them.  IR, network and serial control was at the heart of the original project (along with an emeded ruby engine for scripting them), it will be interesting to see how they implement this again only using lua in a simplifed manner.
