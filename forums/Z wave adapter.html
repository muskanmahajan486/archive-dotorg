Other sticks have not been tested so can't say if it will work – if it implements the same serial API in theory it could but there's always possibility of implementation differences, or altogether different API implementation. OpenRemote does not expose the include/exclude functionality of the serial API yet.
The Aeon stick has a battery and extra button where you can include devices.
You will have to use some other software to perform these steps before trying OpenRemote.
