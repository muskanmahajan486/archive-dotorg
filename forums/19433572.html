When you call "java -version" and you get the correct message, this means that java is within your path.
It does not yet mean that you have JAVA_HOME. The PATH environment variable and the JAVA_HOME are two different once.

If you have eg. Java installed at "c:\Program Files\Java-1.6", you will have a PATH entry which says "c:\Program Files\Java-1.6\bin".
Your JAVA_HOME should point to "c:\Program Files\Java-1.6".
