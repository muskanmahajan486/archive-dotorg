Hi,

Basically i have to take the time to review it and prepare for consumption.

Some stuff is already on sourceforge at: http://m-stack.svn.sourceforge.net/viewvc/m-stack/
More will be added when I'll build some more modules for my apartment.

The 6loWPAN stack used is our own: http://www.modula.si/mstack/en (also to be opensourced)

The whole network is decentralized 6loWPAN with mesh routing on 2.4GHz with a REST gateway in Java.
Every node has also a macro engine with ability to add action to events. ( for example a touch event on a touch switch can toggle a property on a lamp). High level protocol between the nodes is based on wbxml, so is the macro engine.
