Hi Mike,

I've never found a definitive list of DPTs but if you are using ETS4; if you select a communication object of a device in the properties panel you will see a Data Type list allowing you to configure the exact DPT for that communication object. Anyway I have a lot of temp values in my KNX setup and the DPT you want for that is 9.001 (2-byte float temperature degC); in fact any of the 9.* DPTs will work for floating temp values the 3 numbers after the decimal indicate the units but OR just processes the value not the units so in fact which 9.* DPT you use will not matter.

As for your 4-byte DPTs I don't have any devices that have 4-byte DPTs. Although I do have a inZennio Power monitor waiting to be connected up when I get a chance which by the sounds of it will likely use 4-byte objects as well. As 3-byte objects use DPTs 10.* & 11.* I might guess that 4-byte objects use DPTs 12.*; but if you get your device loaded into ETS you should be able to find this out as explained above.

Rich
