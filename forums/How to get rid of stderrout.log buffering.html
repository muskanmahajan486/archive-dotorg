Forums : How to get rid of stderrout.log buffering
This page last changed on Aug 21, 2014 by druciak.

I have replaced line #285:

    local REDIRECT="| head -c 50000 >> \"$CATALINA_BASE/logs/container/stderrout.log\" 2>&1 &"

with:

    local REDIRECT="> \"$CATALINA_BASE/logs/container/stdout.log\" 2> \"$CATALINA_BASE/logs/container/stderr.log\" &"

If you prefer to have one file removing head command should also work:

    local REDIRECT=">> \"$CATALINA_BASE/logs/container/stderrout.log\" 2>&1 &"
