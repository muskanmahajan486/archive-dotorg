Hi Keith,

If I'm correct, the synology package doesn't include the Openremote pro version. The synology package is only made for the standard version of Openremote. The controller in that standard synology package is pre-configured to listen to port 18581.
If you want to use the pro version, you have to update the deployed packages of your syno with the pro packages.
Have you installed the pro version of the controller in your Syno?
