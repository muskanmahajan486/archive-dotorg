If your cable needs are more modest, RS Components sell 100 metre reels: http://uk.rs-online.com/web/search/searchBrowseAction.html?method=browseSubRange&Ne=4294953839&N=4294735965&productNum=0382898.

KNX TP has a flexible topology, almost anything goes except loops. Most devices can support a fan-out of 3 so there is scope for reducing the total wiring length by daisy chaining. The disadvantage for domestic wiring is that the cable has a large cross section. Chasing one cable into a solid wall is messy. More than one, more messy. Wago have 8-way connectors: http://www.wagocatalog.com/okv3/index.asp?strBestNrID=2430208&lid=2&cid=15 and holders: http://www.wagocatalog.com/okv3/index.asp?strBestNrID=2430112&lid=2&cid=15 which can be used for junction boxes with a fan-out of 7.

The KNX cable specification is probably high enough that if you decided against using KNX you could probably run most other bus systems over it (the same cable is used by Wilo Geniax http://www.geniax.de to control and power their miniature pumps).

Radiocrafts have a family of miniature KNX RF modules http://www.radiocrafts.com/index.php?sideID=537&ledd1=535 which might increase its popularity. Calao Systems have a neat USB key device that uses one of these modules: http://www.calao-systems.com/articles.php?lng=en&pg=5990.

The great thing about standards is that there are so many to choose from.

I, too, think that £1000 would be a challenging target for an intelligent domestic heating control system. Note that in the UK you may be able to take advantage of reduced rate VAT (5%) on heating controls for low energy systems which might help.

If you are looking for an entry level KNX system then Siemens Synco Living (http://www.siemens.com/Syncoliving) might be worth a look. This is mostly based on KNX RF (with the potential for extension with KNX TP). It also appears to be a plug-and-pray system where basic configuration can be done without the need for programming by an expensive consultant using KNX ETS.

If your heating system is Viessmann they have Vitohome 300 (http://www.viessmann.com/com/en/products/Data_communication/Vitohome_300.html) which appears to be mainly badge-engineered Synco Living with some Viessmann specific parts (I have seen suggestions that the firmware may be different). Don't know if Vitohome 300 is available in the UK.

The above is based on reading manufacturers' brochures rather than direct experience so please don't commit large sums of money before doing your own research!
