Posted by mredeker at Nov 07, 2012 18:26

My understanding is, that you need the JAVA JDK (Java SE Development Kit), NOT the JRE Java Runtime Environment. (From your directory path I assume you have the JRE). I did an installation under Win7 a few weeks ago:

1) Go to http://www.oracle.com/technetwork/java/javase/downloads/index.html. Download and install the Java JDK for Windows x86.

2) You then need to go to the Windows menu for environment variables under <system><environment variables> and click on in the upper user variable section on <new>.

3) You enter JAVA_Home as variable name and the path to your Java Runtime directory (C:\Program Files\Java\jdk1.7.0_07 in case of JDK version 7 and in case you have not changed the installation directory)

4) In the terminal window change to the the bin directory of your OpenRemote files structure and enter

openremote run

Hope this helps.
