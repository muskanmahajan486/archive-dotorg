Ok, so first unzip the file somewhere on your filesystem, maybe into your C:\ directory.

You can do this with WinZip or similar (Vista has its own unzip I think too).

Just use the 'extract' to get all files inside one directory.

Then you need to find the DOS command prompt – In WinXP, it is under "Start"->Run and then type "cmd".

Then change to directory where you unpacked the files, for example "cd Controller-2.0.0_Alpha11"

And further change directory to "bin" --> "cd bin"

Once in that directory, type "openremote run"
