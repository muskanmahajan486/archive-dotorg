Hi,

I'm a java developer/consultant working on enterprise applications for large companies
in Belgium. My professional experience mainly consists of custom software development
for some large companies in Belgium (banking, public sector, automotive...). Projects
included web applications, J2EE apps ,an MQ JMS framework,..
I don't have any professional experience with home automation: just personal interest.
The electrical system for my house is done using KNX components (mainly for lights).
I've done most of the installation and programming myself, so I have some hands on
experience. My current system is quite straightforward, so mainly just standard push
buttons. I did already do some investigations to extend my system (to use a fancy
touch screen panel or web based remote control). Unfortunately the existing open
source solutions that exist for KNX don't seem very active nor complete (Calimero,
KNXatHome, other TU Wien stuff).
Some time ago I discovered the openremote project and it does seem to be matching
the things I want to accomplish and it's compatible with my current hard- and
software. (KNX components, ubuntu home server)  It even gives me a good excuse to buy
an iPod touch.
The progress the last few months on openremote is also quite impressive, and I can't
wait to try some stuff out myself.
So if you can use my help, I will be happy to contribute, either by writing code or
testing on my home installation.
