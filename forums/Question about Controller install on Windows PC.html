Create a batch file and add the file to start-up.

@ echo off
cd\OpenRemote\bin
start /min openremote run

1. Copy the above code into a text file and save it with file extension .cmd
2. Add this file to your start-up.
3. To Account for power failure change the BIOS settings to restart computer.

Imp Note : The above code presumes that your files are in C:\OpenRemote directory.
