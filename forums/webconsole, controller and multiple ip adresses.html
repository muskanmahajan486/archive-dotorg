Tomcat by default will listen on all NICs but you can get it to bind to a specific one via the Tomcat configuration file (controller/conf/server.xml), see here.

Adding the address attribute to the Connector element in the config file will fix the HTTP listener to a specific NIC.

The auto discovery would need some minor code changes to bind to a specific NIC, this would require some minor code changes to: -

org.openremote.controller.net.IPAutoDiscoveryServer.java
org.openremote.controller.net.IPResponseTCPClient.java
