Dear Klaus,

but sure that you only check those groups as readable which you really want to read on startup. Else you could get those unwanted effects.

The mechanism is as follows:

Object A answers to groups 1 and 2
Object B answers to groups 1 and 3
Set 1 to on
Set 2 to off
Set 3 to on
A is now off and B is on
If you read group 1 now, A will return off and B will return on. Depending on which device answers faster, B will switch to off.
