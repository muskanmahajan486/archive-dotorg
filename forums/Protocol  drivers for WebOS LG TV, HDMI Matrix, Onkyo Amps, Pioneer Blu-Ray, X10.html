This page last changed on Feb 07, 2015 by chewy.

I have over the last few months put some work into developing protocol drivers for the following:

WebOSTV (LG TV) protocol
eZi-HDMI 4 port Matrix
EISCP protocol Onkyo (Improved with sensors)
Pioneer BDP protocol for BDP-140/BDP-52FD/BDP-53FD
CPUXA (X10 ADICON 2500 range) protocol (interfacing with software cpuxad)

I decide to go the specific protocol path as I found it easier and more efficient to handle the sensor side having specific protocols, and you do not need such tricky regex logic to make it work. At the moment they only talk to one device.

I am happy to share this if anyone is interested, what is the best way to share this around?
