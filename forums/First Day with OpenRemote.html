OpenRemote is a integration and visualisation platform. It mediates between devices with potentially different communication protocols via a server (controller) to different consoles (panel) like android, iPad, etc. Devices are further configured (Buiding modeller) to specify their commands, status, etc. Modeler is the name for the program in which you configure all the devices, and the User Interface Designer (Panels)

The minimum configuration is just a PC, (or any other box on which you can install and run Java). 1)Make a virtual switch using the In memory command. Next in UI Disigner configure a switch. Do select two different images, so you can see the results in webconsole
