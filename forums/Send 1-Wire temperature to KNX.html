All,

if you look for a low price solution to send your room temperature on the KNX bus, you can use OpenRemote and 1-Wire.
Just create a custom sensor which reads the 1-Wire temperature and use a rule to have the value send to KNX.

Here is an example:

package org.openremote.controller.model.event
global org.openremote.controller.statuscache.CommandFacade execute;

rule "send temp to knx"
when
  $tempSensor:CustomState( source == "Temp1Sensor" )
then
  execute.command("SetTempCommand", $tempSensor.getValue());
end

Temp1Sensor is my custom sensor which is using a 1-Wire command to read the temperature.
SetTempCommand is a KNX "temp" command which writes the received value as DPT9.001 to the KNX bus.

Much cheaper then "real" KNX temperature sensors

I create a more detailed How-To for this: OpenRemote 2.0 How To - Translate 1-Wire to KNX
