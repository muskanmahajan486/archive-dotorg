Nokia announced yesterday 3 new models, all of which sport a nice looking capacitive AMOLED touch screens:

http://www.pcworld.com/article/205410/nokia_unveils_three_new_smartphones.html?tk=hp_new

If you're a Nokia fan and looking to use OpenRemote with Nokia smart phones, we've got something to get you developer types started with. While we don't have a dedicated Symbian client at this point, we do have a web user interface geared towards mobile clients and have done some preliminary testing with Nokia N86.

Developers can access the mobile web client in the Web Console branch. Notice that this is still early alpha quality so expect to struggle through with it for now. You will also need a non-integrated feature branch of Controller which contains a new JSONP API (available here).

Good luck and let us know if you find something terribly amiss. We will be integrating both the web console and JSONP API to main release branches later.
