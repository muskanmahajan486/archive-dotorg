Posted by mredeker at Apr 22, 2009 15:08

Is it possible to test with my iPhone and irtrans module already? Let me know how I can install.

At the moment we send IR directly to LIRC. It's however just a delegation to LIRC's irsend command via Java's ProcessBuilder. Search for IRCommander.java in Controller tree.

So basically you'd need to hack that to talk to the IRTrans software daemon (if I understood you correctly) that translates to socket and IP for your IRTrans unit. There's a "Commander" class to extend for this. I'm working through these same steps as we speak myself as well.

That would get you to the point where you can poke the controller with a very simple REST command to trigger an event (see Controller API Documentation). Your controller will also need a controller.xml that maps the incoming REST cmd paramter (numbers 1,2,3 etc) to an event that will direct to the "Commander" class.

So still some hacking required (but we haven't released yet so...)
