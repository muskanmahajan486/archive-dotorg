Hi Andrew,

everything started fine (I guess we could/should add a message to the log to be more clear about it)... I assume the shutdown messages didn't trigger automatically but you closed the process.

So after the startup, you can point your web browser to http://localhost:8080/controller

In the home page you can enter your OpenRemote Designer username and password and download your design into the controller (that will bring your missing controller.xml file into place). You'll need to create a separate designer application account first though (it's not the same name and password you used for these forums) and you'll need to put a few buttons in place and create a UI panel.

HTH
