Lung, welcome to OpenRemote.

You need a PC running the controller part of OpenRemote. The iPhone, Android panels connect to this controller to tell it what todo. If you eg want to turn on a light the iPhone will send a command to the controller and the controller connects to your lighting system and tells it to turn the light on.

The internet link is needed to create the system design in the online modeller. When this is done the controller syncs with the modeller only once (or if you change something) to download the design. After that no internet connection is needed anymore.
