Community members have posted QNAP instructions online, see here:
OpenRemote 2.0 How To - Controller on QNAP NAS

As for the connections, you'll still need the mux daemon if you have more than one application connecting to the KNX gateway (your QNAP NAS with OpenRemote Controller would count as one).

If that is the case, you still have to install the multiplexer somewhere and have controller go through it. It looks like QNAP NAS is fairly open Linux system so you may be able to install eibnetmux daemon there too.
