Posted by tomtibo at Oct 19, 2014 19:31

Controller is calling back to cloud because of features such as auto-discovery (e.g. for Z-wave devices and in the future remote proxy).

Check the config.properties file (under WEB-INF/classes), where you can set 2 properties to controller this:

controller.performBeehiveSyncing=true, set to false if you don't want any call back to cloud

beehiveCommandService.check.interval=5000, set to desired call back interval, in milliseconds
