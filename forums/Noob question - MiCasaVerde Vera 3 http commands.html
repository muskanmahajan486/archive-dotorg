I get one thing working. I can switch on a socket plug with the following command:

HTTP, Get command:

http://<IP-ADDRESS-VERA3>:3480/data_request?id=lu_action&output_format=xml&DeviceNum=<DEVICENUMBER_OF_SWITCH>&serviceId=urn:upnp-org:serviceId:SwitchPower1&action=SetTarget&newTargetValue=1

To switch it off it should be (I guess):

http://<IP-ADDRESS-VERA3>:3480/data_request?id=lu_action&output_format=xml&DeviceNum=<DEVICENUMBER_OF_SWITCH>&serviceId=urn:upnp-org:serviceId:SwitchPower1&action=SetTarget&newTargetValue=0
