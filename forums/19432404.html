I am a newbie too but I do a best effort at anwering;

1. There is a online designer where you create your design. The design includes a graphical UI for a remote and configuration of your devices. It maps the controls of the UI to commands of your devices. The design is uploaded to the controller. After the design is uploaded devices connected to the controller can be used as remotes using your ui design.

2. There is protocols implemented for a number of devices, including HA systems (Russound, Lutron). New protocols is added constantly. The ones you mention are not supported, to-date.

2.1. Yes. There is the 'console' project where you can use any browser for device.
2.2., 2.3. There is so far practically no support for programming or scripting in the designs created using the tooling.

3. They are downloaded from the controller.

4. Well. You should really start with looking what is there. Make sure to make notes on what may be improved so that when you know your way around you can either suggest improvements or, if you apply for an editor account, you may update the documentation yourself.
