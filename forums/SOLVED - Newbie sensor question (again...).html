The regex depends of what the output is of your shell command. If it is just digits then (\d+) will be enough. "Custom state items" you leave just empty. The one thing you should remember, because there is a glitch in the shell command protocol implementation you must reboot the controller each time you change something in designer and upload it. Otherwise you will be still using the old cached command. This can give you a false impression that nothing is working.
