Is there anyway to have telnet UI composer or XML specify the user and password in additon the commands.

For example I want to telnet to a server as root which requires user: root and password: xxx then to run a command.

From a console I can do:

machine$ telnet 192.168.1.1

sever login: user

server# echo -e  !1PWR00
r>/dev/ttyUSB0

Comments:

When you create the telnet command in the UI composer you have to specify the string in the format: <waitFor>|<send>|<waitFor>|<send>

So your example would have to be configured as: 

login: |user|word: |xxx|server# |echo -e !1PWR00R>/dev/ttyUSB0
You just cannot have a "|" in your code, because we don't support escaping yet
