Do you already have ipkg installed on your Synology? http://forum.synology.com/wiki/index.php/Overview_on_modifying_the_Synology_Server,_bootstrap,_ipkg_etc#How_to_install_ipkg

If yes, nano is a fairly simple text editor to use from the bash command line. You can try:

> ipkg install nano

Then you will be able to edit the file with:

> nano /usr/syno/etc.defaults/rc.d/S99Modules.sh
