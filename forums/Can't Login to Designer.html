Posted by doucga at Sep 19, 2011 04:16

There is a configuration which is used to create the link in the email. If you have this running locally maybe that link does not point to the correct server? I think it's this property in config.properties of the modeler application:

webapp.server.root = http://localhost:8080/modeler/

You can also activate the user manually when you set the attribute "valid" in the user table of beehive to true.
