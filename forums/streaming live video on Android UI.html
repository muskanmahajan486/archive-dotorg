Hi,

Browsing on Internet I've found someone experiencing the same issue:

http://stackoverflow.com/questions/12708890/youtube-video-not-playing-in-webview-android

and getting a solution using setWebChromeClient method of WebView class:

wv.setWebChromeClient(new WebChromeClient() {
});

From WebView Documantetion:
In order to support inline HTML5 video in your application, you need to have hardware acceleration turned on, and set a WebChromeClient. For full screen support, implementations of onShowCustomView(View, WebChromeClient.CustomViewCallback) and onHideCustomView() are required, getVideoLoadingProgressView() is optional.

Was wondering if this could fix the issue in Android Console.
