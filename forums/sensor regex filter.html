Which sensor are you using in OpenRemote?
Please try and use a custom sensor with no extra items and make your label field big enough so it can display the three lines.

If you want only the '48.60' as a value, you will have to change the regex to only return one matching group since the current code only returns the first match. You will still have to use a CustomSensor for the 48.60 though, since we do not have a floating point sensor.
