The issue was with installing the newer version of the Synology DSM 5.0 I had to reinstall java as well.
I did not remember that java version 7 was not supported.
So I removed java version 7 and installed Java SE for Embedded 6u38 (ejre-1_6_0_38-fcs-b05-linux-i586-headless-13_nov_2012). You can find it here : http://www.oracle.com/technetwork/java/embedded/downloads/javase/default-1934076.html

I am using a Synology DS1813+

After restart of openremote it all worked.
