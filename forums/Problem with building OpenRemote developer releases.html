Comments:

Using workspace branches without asking first is generally not recommended, as these really are workspaces and therefore may not compile, or may include code that will never be included in the releases.

That being said, try 'svn update'.

You will find the stable release tags here : https://openremote.svn.sourceforge.net/svnroot/openremote/tags/project/
