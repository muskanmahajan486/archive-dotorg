I've added a macro for embedding video content into blogs, wiki pages, etc.

It currently only accepts YouTube as the host, and has a fixed size. If you need different size options or different video host sources, let us know here on the forum replies.

You can use the macro like this:

{video:id=JYFDB5ao2v0}

Where the 'id' parameter is the unique video id in the YouTube URL (in this case, http://www.youtube.com/watch?v=JYFDB5ao2v0).

If you're using the "Rich Editor" to create pages or forum posts, you need to switch to "Wiki Markup" to add the macro as shown above.

Any questions or comments just post it here in this forum.
