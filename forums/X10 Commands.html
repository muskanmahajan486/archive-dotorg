Posted by mgriff2000 at Apr 07, 2014 01:55

Supported X10 commands:
ON
OFF
DIM
BRIGHT
ALL_OFF | ALL_UNITS_OFF | ALL OFF | ALL UNITS OFF
ALL_LIGHTS_ON | ALL LIGHT ON | LIGHTS ON

All commands are case insensitive.

As a command parameter use device address House_code Unit_code, for example "DIM A3".
See http://sourceforge.net/p/openremote/code/HEAD/tree/tags/project/Controller/Controller_2_0_2/test/org/openremote/controller/protocol/x10/X10CommandBuilderTest.java
