I assume you mean a simple one-way command, and not a switch?

If that's the case, create a command as usual (KNX group address, and command such as 'on' or 'off') and then on the UI designer side create a button instead of a switch and map the command to this button.

Note however that if you need a specific scenario value 1..n then that's not included yet, each scenario would have to be in its separate group address with a boolean on/off (0..1) value.
