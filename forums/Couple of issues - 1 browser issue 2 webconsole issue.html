Posted by timbocephus at Sep 11, 2012 02:51

Hi Tim,

It's hard to tell if you've got the latest version by looking at the file names but OpenRemote-Controller-2.0.0.zip is the download name of the current controller download as advertised on the website so I'd make a guess and say that is the latest.

As for instructions; the fact that everything is JAVA based means there shouldn't be any issues running across multiple platforms and I have no problems running the same controller from Windows or Linux (Ubuntu). But the basic steps for Ubuntu would be: -

  • Open a Terminal Window
  • Type 'sudo apt-get install openjdk-7-jre' (without quotations)
  • Type 'sudo update-alternatives --config java' (without quotations) make sure the openjdk-7-jre is set as the default
  • Type 'java -version', you should see some text with 'java version "1.7.0_147-icedtea' in there

You now have JAVA JRE configured.

To run the controller, unzip the files somewhere then cd into the directory and then type 'sh bin/openremote.sh run' (without quotations) and you should see it running with messages appearing in the terminal window.
