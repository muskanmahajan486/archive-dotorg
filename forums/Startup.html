Hi

That's fairly easy to identify.

The demo / free version of OpenRemote controller uses port 8080 - so needs a demo / free designer account, which can be found at composer.OpenRemote.org

Whereas the Pro version of the OpenRemote controller (by default) uses port 8688 - so needs a Pro designer account, which is accessed via designer.OpenRemote.com

So, if you have a Pro account, please log into the Pro designer and download the Pro Controller using a button on the top right hand side marked "Download Resources"

Good luck
