Hi,

The "Data Link Layer General" describes general concepts about this layer.

Here , general means not specific to a physical media (twisted pair, powerline, IP, RF, ...).

The layer provides the medium access control and the logical link control. The logical link control examines the frame format and
destination address of a frame and decides if the frame shall be received. The data link layer
detects transmission errors and repeated frames.

The details for twisted pair medium are described in KNX  Volume 3 part 2 chapter 1 section 2

Hi,

There is no layer called "Data Link Layer General". The layer communicating with network layer is "Data Link". The document called "Data Link Layer General" describes the general aspects about data link layer, and all media specific details are (for twisted pair) in the document I referenced above. They both speak about different aspects of the same "Data Link" layer.
