Forums : MinutesTillSunrise off by 2 hours
This page last changed on Feb 08, 2014 by aktoy098.

I implemented a sunrise/sunset set of rules for managing my lights. I added 2 sensors under the datetime device. Both being the minutesTill... type sensor. Both are set for my location 39.3,255.2 and UTC-7. I put a label on my panel showing the values of these sensors. The minutesTillSunset returns the correct value for sunset. The minutesTillSunrise returns values that are exactly 2 hours off (later). Very strange. So I adjusted my rules to account for this discrepancy and they work well.
