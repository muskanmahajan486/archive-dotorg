Hi - Wunderground is excellent for xml weather data - heres an example of current temp...

Create a command...
Name : Wunderground_Temp
Protocol : HTTP
Command URL : http://api.wunderground.com/api/[userkey]/conditions/q/pws:IBRISTOL29.xml
HTTP Method : GET
XPath Expression : //response/current_observation/temperature_string
Polling Interval : 10m

Create a sensor...
Name : CurrentTemp
Command : Wunderground_Temp

Create a label...
Sensor = CurrentTemp

Also have a look here http://www.openremote.org/display/docs/OpenRemote+2.0+How+To+-+HTTP+Protocol

HTH
