I have a DaliControl SC 64 IP from IPAS. This is a KNX DALI gateway capable of controlling 64 lights. I am currently experimenting with one Light.

I was experimenting to see if I can use this device only with network connection (without KNX) to control a light with DALI LED Driver.

I am able to paste the below link in browser and turn off the light where 192.168.1.112 is the IP address of the device.

192.168.1.112/dm?x=F6C351013B000100010B0000

I am able to modify the last two digits to FF and turn on the light the command is as below.

192.168.1.112/dm?x=F6C351013B000100010B00FF

I have also tried in with various numbers like 0000, 0003, 0005, 000A ...... 00FF values from 0 to 255 to acheive various dimming levels.

Is there a way that I can use OpenRemote to control the light? Maybe using http Commands?

Below is some info I obtained using Wireshark.

GET /dm?x=F6C351013B000100010800FF HTTP/1.1Host: 192.168.1.112
