Controlling Humax FOXSAT-HDR Freesat+ 320GB HD Digital TV Twin Tuner HD Recorder
With OpenRemote.

Download and Update the firmware on the Humax FOXSAT-HDR using this link

http://www.avforums.com/forums/freesat/1661195-media-file-server-bundle-foxsat-hdr-release-4-0-part-3-a.html

Once installed log on Humax box using your web browser

Download and install the full web interface and the remote control application.

Once you have completed the above enter the following into your web browser

http://192.168.1.103/plugin/remotecontrol/cgi-bin/remotesend.jim?signal=CHAN_UP

If everything is installed correctly this should change the Chanel.

Now open up Designer and start entering commands

Thanks for the command set and command line syntax. I've used this with great success. However one command does not work for me - OPT+. If I experiment with this in a normal browser window, the command is returned with an error because the + has not been sent. It appears when I try to execute the http request for this command, the + is dropped.

Any suggestions?

Edit to add:

Have solved this issue from another forum. Instead of entering OPT+ as the command it needs to be sent as an escape sequence ( A % symbol followed by the ASCII hex value of the character). Therefore if entering the http command for OPT+ in a browser or through a third party app, use OPT%2B.
