Comments:

This feature has been implemented in some branches but not been merged to the main release.
The problem being that this feature does break backwards compatibility and requires all components to be updated at the same time (modeler, controller and consoles).
We're targeting to integrate this after a 2.0 release of all components has been done.

If you want to try this out, you'll need to build all the components yourself (including the iOS app) from the repository. Look for the branches with _ButtonEnhancement in there name, mainly under workspace/ebariaux.

When building the iOS app, you need to define "API_v2_1" so the correct version of the API will be used.
