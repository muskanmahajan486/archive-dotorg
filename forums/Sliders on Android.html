Ok, made another crude patch for the out-of-range issue:

In knx/ApplicationProtocolDataUnit.java:

@@ -667,7 +668,14 @@
      */
     private Unsigned8Bit resolveTo8BitValue(DataPointType.Unsigned8BitValue dpt, int value)
     {
-      return new Unsigned8Bit(dpt, value);
+      int newvalue = value;
+
+      if (value < 0)
+      {
+        newvalue = 255 + value;
+      }
+
+      return new Unsigned8Bit(dpt, newvalue);
     }

With this patch, and the one above, I got both the slider and label to work.

Not suggesting that these two patches are the correct fixes, of course, but maybe they can help with narrowing in on the issues.
