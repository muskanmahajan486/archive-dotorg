Hello,

I found a nice way to send push messages to my smartphone.
I use the app "Pushover".

Step 1.
Buy and install the app from the appstore.

Step 2.
Get your token from their site.

Step 3.
Create a .sh file wich can send messages via the pushover api (/home/pi/pushover.sh).

curl -s \
  -F "token=yOuRTokEN" \
  -F "user=youRuSer" \
  -F "message=$*" \
  https://api.pushover.net/1/messages.json

Don't forget to make ik exacutable. ( chmod +x pushover.sh )

Step 4.
Create your command in OpenRemote.
