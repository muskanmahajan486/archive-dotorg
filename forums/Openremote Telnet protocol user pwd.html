The command string of the telnet protocol supports a very easy handshake mechanism:

/**

  • A pipe separated list of command strings that are sent over the connection It must have the format
  • <waitFor>|<send>|<waitFor>|<send>
    */

If your host asks you for a login the telnet prompt looks like "login:" and you can use that as <waitFor>.
Then comes something like "password:" and you can use that again.

The command for the telnet protocol would look like: "login:|username|word:|password"
