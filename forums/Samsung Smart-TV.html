Posted by mredeker at Oct 24, 2011 10:12

Based on the Android App from Tom Quist, I added support to remote control Samsung TV's through network.
The code is available in my current developer branch: http://openremote.svn.sourceforge.net/svnroot/openremote/workspace/mredeker/Controller_EP_SNAP_DateTime_WOL

Auto-Discovery with UpNP is not yet supported and you have to configure your TV IP-address within config.properties.

The protocol tag is called 'samsungTV' and you need to configure the commands with one property 'keyCode' which takes a String like:

  • KEY_CHUP
  • KEY_CHDOWN
  • KEY_VOLUP
  • KEY_VOLDOWN
  • KEY_MUTE
  • ....

All supported keycodes can be found here: https://openremote.svn.sourceforge.net/svnroot/openremote/workspace/mredeker/Controller_EP_SNAP_DateTime_WOL/src/org/openremote/controller/protocol/samsungtv/Key.java

Maybe not all of the codes will work for your TV. Mine eg. does not support "KEY_POWERON" but maybe others do.
