You will need a XAMPP version that includes Tomcat. The OpenRemote controller needs a Java servlet engine which Tomcat provides.
If you have XAMPP and Tomcat running, you just need to take the folder "controller" from our distribution and copy it to the webapps folder of the XAMPP Tomcat.
