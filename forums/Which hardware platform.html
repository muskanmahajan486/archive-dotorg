Operating system is going to depend slightly on the kind of integrations you're looking for. Something Intel Atom based is usually sufficient to run OR so if you have PC boxes running, I'd start with that. Once the system is running like you want, you can always later transfer the controller setup to a smaller system like RPi or Synology – the latter two always require a few extra steps to get things running so you might want to eliminate too many moving parts if you're just getting started.
