Posted by crvchul at Jun 15, 2013 08:47
  • Z-Wave>Me only does Z-Wave devices, has a limited user interface, but does include support for a great many devices.
  • You can set up your network of Z-Wave devices using Zwave>Me's Z-Cloud service, and next use that setup with OpenRemote, provided that you use the same controller (Z-stick). You must take care to shutdown the service you are not using.

For the Danfoss Radiator Valve search this forum for "DLC" or "Living Connect". Be warned that Danfoss has had the wisdom to start selling non Z-Wave compatible versions of this device under almost the same name.
