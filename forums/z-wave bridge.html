Posted by orcking at Feb 12, 2012 17:31

The Vera most likely is more powerful then the HSC-40. The HSC only supports basic z-wave command classes in the moment.
The door lock command class is pretty sophisticated and supports a lot of stuff.

OR sends HTTP commands to the HSC-40, yes, but the status is reported in a rudimentary binary format for which we created some code to be able to give status feedback.
Unless HSC supports more z-wave command classes and provides a more solid protocol, I doubt we can support door locks.

Supporting door looks with our native z-wave support will be possible but that is a little way down the road.

Posted by orcking at Feb 13, 2012 00:07

Sending HTTP "get" commands is already possible with OR. So you probably can already turn lights on/off with Vera and OR
The HSC is sending the status when something is changed. I need a TCP socket for that to listen to changes.

OpenRemote Inc. (the company behing OR) licensed the z-wave API and we are looking into native z-wave support.
How that will work and what we will support is not finalized yet. Also how this feature will be integrated is still open since we cannot publish any z-wave API related code.
