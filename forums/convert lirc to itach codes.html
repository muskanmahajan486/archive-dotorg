Comments:

You can use my Lirc2Xml program to convert the lirc files to CCF codes embedded in XML. Converting CCF codes to "sendir format" is in principle easier, there are probably programs around, or it is rather easy to write one. (I also have stuff, but unfortunately presently in releasable form).

Unfortunately, there showed up a bug in the old version 0.1.2 of Lirc2xml (the repeat sequence was the only thing that came out for NEC1-like signals, like the ones for Topfield), which I fixed in the just released version 0.1.3. So be sure to use 0.1.3 for NEC1-signals.
