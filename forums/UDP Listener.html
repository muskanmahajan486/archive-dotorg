There is no "just UDP listening" protocol in the moment. The UDP protocol of the controller can be used to send something our via UDP and then wait for an answer. I already have another development branch, which includes a UDPListener only protocol which can be configured to use a RegEx to update the sensor and a port where it listens. This branch is not publicly available but one of the next controller versions will include it.
