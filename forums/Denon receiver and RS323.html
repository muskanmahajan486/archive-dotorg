i have a Denon 2307 receiver which is connected by RS323 and I am able to control all the features except for the slider and set volume commands. Does anybody know how to resolve this issue?

Comments:

The slider can not work, since the protocol so far do not support reading values.

The issue with the set-volumes is a bug in the command definition. It have been resolved here: http://svn.code.sf.net/p/openremote/code/workspace/toesterdahl/Controller_2_1_0_Integration/config/denonAVRSerialCommands.properties

If you choose to just edit the file, change 'mv.00=MV00 -80' into 'mv.00=MV00'.

'-80' used to be a comment which indicate the display reading, which is -80 dB in this case.

Since the OR Denon Protocol does not have reading implemented you could try using the OR Serial Protocol implementation. Your Denon receiver accept any of the ascii words from the denonAVRSerialCommand.properties as a command. If I remember correctly it also expect a carriage return at the end of the line.
