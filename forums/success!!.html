Hello to all,

Since last night i can proudly say that i have open remote running and controlling one lighting point in my knx home installation!!! (Looking forward to more complex controls) It took some effort but it was worth it!! This forum gave me a lot of guidance so i would like to give some feedback in order to help others as well.

My setup is as follows:

knx IP network device: Eibmarkt N000401

open remote controller running on Windows 7

Antroid device Sony XPERIA ST23i phone (Android 4.0.4)

Network settings set to "HOME NETWORK" (i think this setting helped with some communication issues)

Line couplers in the knx installation programed so that they do not filter any telegrams

Status group adress to be different to the command group adress and to contain the status object of the device you want to control.

Internet provider modem-router and my wireless home router setup to have similar ip adress.

Best Regards,
