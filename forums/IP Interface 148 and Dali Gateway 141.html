You can try to have a look to the Calimero API (java), it is used in OR. Otherwise, there are stacks available, some commercial, a fewer opensource.
If you want to go the hard way, you can look for KNXNet/IP doc, there are a few explanation on the web. Then, you can simply open a socket to the gateway and send the correct raw byte to the gateway.
