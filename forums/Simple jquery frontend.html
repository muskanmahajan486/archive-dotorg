Hey guys, I wanted to play around with jquery + axis2 + jomnilink (my ha library) and came up with a proof of concept web front end over the weekend.  It's actually fully functional, although I have disable command functions (so the public can not turn my lights on and off).

The javascript and html was simple to write, there really is not a whole lot (view the source) to it.  I also used a wait and hold approach to querying web services (ala a comet like system), so requests will hold up to a certain time if no new data is available.  This means if a light, thermostat, zone, etc changes it's imediately updated in the the web interface.  I really like the idea of exposing a system like OR via SOAP and REST. I'll keep it up for a few days.

Let me know what you think.
