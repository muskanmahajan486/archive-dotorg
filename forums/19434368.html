See the attachment for the snapshot of Richard Turner's version 2.0 (Alpha) of Web Console from Dec 28th.

You will need to get the Controller snapshot that Marcus created recently which is here: http://www.redeker-consulting.de/openremote/OpenRemote-Controller-2.0.0_SNAPSHOT_20120103.zip

The 2.0 alpha of WebConsole won't work with older versions of the controller.

Drop the webconsole2.war file in the controller's webapps directory. The webconsole should show up at http://localhost:8080/webconsole2

The initial config screen is still work-in-progress but enter the controller URL (http://localhost:8080) and your panel's name (which you entered in the designer). You'll have to sync your controller with the designer first to make the panel available, which you can do at http://localhost:8080/controller
