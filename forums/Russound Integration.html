Posted by spencerthomass at Dec 22, 2011 15:40

First you have to configure your connection parameter in webapps/controller/WEB-INF/classes/config.properties. The valid commands can be found in the file org.openremote.controller.protocol.russound.RussCmdEnum.
The Power and Loudness commands need a sensor of type switch.
The Volume and TurnOnVolume a range sensor 0-100.
The Balance,Bass,Treble a range sensor -10 to 10.
I am currently working on a wizard for the designer which will generate all of that for you.
