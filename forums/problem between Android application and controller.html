Going back to Pieter's original question about webconsole (to isolate the issue you're seeing to Android only).

Can you try the bugfix version of webconsole (build 8895) mentioned here: Bug in webconsole

Steps:

1) stop controller
2) go to Controller/webapps directory
3) delete webconsole directory
4) delete webconsole.war file
5) download and save the build 8895 from the above-mentioned thread and save it to Controller/webapps/webconsole.war file

Would be good to get at least one console working to troubleshoot your issue.
