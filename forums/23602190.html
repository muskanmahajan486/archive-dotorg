In August, a note was added in the forum about a next release. 

An upcoming version of the new Z-Wave protocol (see New Z-Wave 2.0.0 Release) will have support for the Fibaro RGBW Controller and other devices that have support for COMMAND_CLASS_COLOR_CONTROL (Zipato RGBW Bulb, Aeotec LED Bulb). That means it will be possible to use them in combination with the color picker control.

I've managed to add the OpenRemote controller as secondary controller by means of a Vera controller but Z-Wave network replication from primary to secondary controller is still missing because I'm currently working on "more important" things like Z-Wave door locks and the related network security layer.

So stay tuned.
