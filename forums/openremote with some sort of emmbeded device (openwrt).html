The OR consoles (iOS / Android apps) at this stage are meant to only work with a controller and there is no way to directly control devices from them.
For KNX installations, there is an ORB less iOS application (found in the AppStore) but this does not seem to be what you're after.

The controller is pure Java so can run on many architecture (where a JavaSE 6 VM is available). But if you don't want to put additional load on your OpenWRT box, then you'll need some other device to run the controller on.
