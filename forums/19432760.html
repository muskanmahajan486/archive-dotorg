Hi,

Short answer: out-of-the-box no.

This will require some healthy tweaking on your part. The controller will understand commands sent over HTTP. You'll need some program or script on the Mac that generates these HTTP requests. Then you'll need something that can take the incoming IR command on Mac and execute your little program or script. One example that might work (have not tried any of this) is here: http://twistedmelon.com/mira/screenshots.html#mainwindow

There are other similar products out there if you google a bit. The HTTP commands OR controller understands are described here: http://www.openremote.org/display/docs/Controller+2.0+HTTP-REST-XML#Controller2.0HTTP-REST-XML-SendWriteCommand
