Comments:

Regarding consoles, we are building these for non-Apple platforms as well. Android is currently under testing and web console should be available soon.

Regarding Windows app, you can build Win32 console that utilizes the HTTP/REST API that we provide.

HA devices, very much an individual choice. We recommend devices that have ability to integrate with an IP-based network (IP-to-IR, IP-to-Serial, IP-to-KNX, etc.).

Haven't had time to look into Insteon support yet (only works in the US) though there's a reasonable bridge there to IP from Universal Devices (ISY-99). Same with Z-Wave – know some IP-to-Zwave products in development but haven't had time to test yet.
