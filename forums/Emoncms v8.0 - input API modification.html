In order to send data to Emoncms it is needed to set the "emonURL" in the XML file "applicationContext.xml" (see How To on Emoncms). Emoncms has recently released version 8.0. In this version the Input API of Emoncms has been modified a little. You now need to also insert a node ID in the "EmonURL" (integer value between 0 and 32) in order to Emoncms v8.0 will accept the input.

"http://[IP ADDRESS OF THE WEBSERVER RUNNING EMONCMS]/emoncms/input/post.json?&amp;apikey=[YOUR_APIKEY]&amp;node=[AN INTEGER BETWEEN 0 AND 32]&amp;json="
