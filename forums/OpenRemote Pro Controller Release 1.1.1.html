A new Pro Controller release is now available for Pro Users.
Version 1.1.1 is now available from the Download window of your Pro Designer account (text still says Pro 1.1.0 but Zip has been updated).

This is a bug fix release and includes fixes to the following issues:

  • ORCJAVA-407: Drools exception when executing rules after controller xml has been reloaded
  • ORCJAVA-451: RRD4J Protocol commands that change the start or end time of a graph also cause the end or start time respectively to be fixed
  • ORCJAVA-459: Add more logging information on connection failure when retrieving controller identity* ORCJAVA-471: Decode request path segments was incorrect
  • ORCJAVA-476: Issue with Restlet not releasing resources appropriately, resulting in controller lock up (Update version of Restlet library used)
  • ORCJAVA-489: Vera: Issue parsing empty strings in status values
