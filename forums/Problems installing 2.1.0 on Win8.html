Dave,
Just be mindful of the fact that your user and password for logging onto the forum site (http://www.openremote.org/display/forums/User+Forum) are not the same thing as the user and password for Open Remote Controller.
You may set them the same but they are not actually joined together.
Neil.
Posted by nwe at Aug 13, 2013 04:03

The User/Password which you can enter at the controller is actually the one you define when creating a designer account.
After setting up a designer account and creating a design those credentials are used by the controller to download the design into the controller.
Posted by mredeker at Aug 14, 2013 09:09

Also make sure you respect case. IIRC, the designer login is not case sensitive, but the controller sync functionality is.
