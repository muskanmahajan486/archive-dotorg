Increasingly, more devices have started supporting network capability. This allows one to control these devices using a browser or an app on a mobile. However, this poses another problem - multitude of interfaces to control the devices. This project aims to bind all these together in a single unified voice controlled interface.

In the following demo, the voice controlled app is controlling a spark core enabled light and Denon AVR.
