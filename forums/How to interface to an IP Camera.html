Posted by aktur at Jan 01, 2014 12:07

Just be careful that once you use the preview or staging designer, you should NOT try to use the standard "demo" designer again.
It you're using a newer widget (like Web view) and try to login in the demo designer, your designs will appear empty.
If the autosave kicks in, you'll now have saved an empty design and lost all your previous work.

But as long as you stick to always using the same designer, you should be fine.
