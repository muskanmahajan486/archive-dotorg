You are not doing anything wrong. The webconsole at present will always load a panel of size 320x480 (iphone size).

The current XML schema for the panels does not indicate the size of panel (as defined in the designer), this is set to be included in the future.

What the web console does need is to remember your preference (i.e. fullscreen or width and height). I will add this functionality soon so keep an eye on the forums!
