shows that the web server starts OK. Therefore you should be able to access http://localhost:8080/controller and sync the design. Check your firewall settings in case of problems. Alternatively, you can put the design in OpenRemote/webapps/controller/ manually.

java.net.BindException: Address already in use: JVM_Bind

It seems that you have already some process using port 8005. You can try to either kill this process or change the port in server.xml
