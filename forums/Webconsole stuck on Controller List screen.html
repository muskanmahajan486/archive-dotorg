Martin/Doogie,

I also have been trying to get openremote going on Ubuntu (10.10). I am new to this and my setup still isn't quite right, but I am able to see my panels from the designer now. These are the steps I took near before I got it to work. It sounds like you have already tried most of these, (your tomcat server is running) but just in case:

1. Goto http://myubuntuhost:8080/controller
2. click on <sync with online designer> button
3. http://myubuntuhost:8080/webconsole
4. click on <search> to find controllers
5. click right arrow for my controller to add panel name (not screen name)
I didn't need to add name and password for controller
6. Finally click on the controller and run

I think there is an "Export" icon to manually update your controller also.
