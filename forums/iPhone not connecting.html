If you controller runs on port 8080 you will have to enter the controller URL in the iPhone like this: <ip>:8080/controller/
You can test if the controller responds correct by just entering the following in your browser: http://<ip>:8080/controller/rest/panels
This returns a xml structure which shows the panels you defined in the designer.
