If you mean the Fibaro Universal Sensor (FGBS-001), I am pretty sure it is not yet supported in the OpenRemote native Z-Wave implementation. The Aeon labs USB stick did work for me with OpenRemote Z-Wave.

DiskStation> cat /volume1/@appstore/OpenRemote/logs/zwave/zwave.log
INFO 2013-10-13 17:58:02,234 (1-WIRE): Reader thread asked to start

This (1-WIRE) thing in the string is a relict which was left from early experimentation when the log settings were copied from the 1-wire protocol. I don't use OR-Zwave anymore, but I thought this had been fixed in the meantime. Do you happen to use an old version of the controller?
