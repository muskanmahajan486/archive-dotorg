Hi Luca,

The message you have seen is a bit mis-leading but the following should solve your problem: -

There are a few ways to set Environment variables in Windows but easiest is to set a system wide environment variable: -

Control Panel -> System and Security -> System then select Advanced System Properties on the left hand side.

Select Environment Variables (at bottom of Advanced Tab) then add the JRE_HOME variable to the bottom section of the window (System variables).

Variable Name: JRE_HOME
Variable Value: Path to the JAVA Runtime (JRE) e.g. (C:\Progra~1\Java\jre6) avoid paths with spaces

Make sure you are not setting the JRE_HOME variable within the batch file used to start OpenRemote as well.
