Posted by pz1 at Nov 04, 2013 22:08

I recently also bought a razberry, here are some things you need to know:
z-way (eg z-wave>me, their naming is a bit unclear) is the software delivered with the razberry and is the (controller) software that comes with the razberry hardware is actually a developers kit which offers an API to develop your own software. The software also includes a demo version to get you started but it doesn't seem to be fully functional (although it is working quite ok for basic implementations)
The API provides a series of REST functions to control your z-wave hardware. REST functions can be called with a simple http request to the ip of your raspberry pi.
No security features are implemented, any one with access to your network can control the z-way controller. It is possible to put the controller (server) behind a nginx webserver and to configure some access control on the nginx webserver but it requires more knowledge and time to configure this.

The wireless range on the range on the razberry is totally unsatisfactory, for me it could not span 5 meters with a tv and some other devices in between. The hardware also doesn't provide a way to add an external antenna.
