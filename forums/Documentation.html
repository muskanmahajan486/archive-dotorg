The 1.0 release goals are explained on the Release Page. It is useful for remote controlling at this stage.

The only remotes included at the moment are iPhone and iPod Touch, as you probably noticed while creating the interface. If you want other means to control, you can help with the web console work David Reines started here: http://openremote.svn.sourceforge.net/viewvc/openremote/trunk/Panels/Web/ or the Android panel here: http://openremote.svn.sourceforge.net/viewvc/openremote/trunk/Panels/Android/
