Forums : Denon AVR Serial RS232 Support
This page last changed on Apr 11, 2011 by toesterdahl.

I have added basic support for controlling Denon AVR equipment over serial RS232. The code is checked in on my workspace https://openremote.svn.sourceforge.net/svnroot/openremote/workspace/toesterdahl/Controller_2_0_0_Alpha11_DenonAVR. I am opening this thread to encourage a discussion around support for Denon AVR's.

The implementation is based on a specification for AVR-2807/AVC-2920.

It appears that the same same commands should work for the following range of equipment, however this is so var not verified.

Model (Denon AVR/AVC) Verification
2106
2803
2805
2807 toesterdahl 2011-04-11
2803
3805
3806
4306
4802
4806
5803
5805

Note that the implementation as it stands is rudimentary, limited to the features it supports and should count as a first snapshot. It might make sense to wait for the implementation to take better shape. At the end it could be that dedicated support for Denon is dropped in favor of generic support for RS232.

You need to configure the denonAVSerialProperties in webapp/controller/WEB-INF/classes.
Once the connection is configuered correct, you can use the commands which are listed in the denoncommand.properties (same folder).
The command name has to be used in the designer when creating a new denon command.
