Other possible reasons are that your OpenRemote controller is not in the same subnet with your IP gateway – therefore the automatic IP detection may fail.

If that's the case, look into OpenRemote directory /webapps/controller/WEB-INF/classes/config.properties, there you will find an empty line for knx.ip address – you can enter your KNX gateway IP address there (a fixed address, rather than auto-discovered one).
