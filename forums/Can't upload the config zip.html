Comments:

Did a little more digging and yes, the file path is all wrong. If I created a path in root like /webapps/controller, it put the config there. I'm not sure what path is wrong, I suspect the CATALINA_HOME, but I can't seem to get it right.

Edit:
I finally figured it out! When I ran it manually using the same user as my startup script I noticed this output:
No directory, logging in with HOME=/

So I changed the home dir of my openremote user to /opt/openremote/bin and now it works!
