To share my experience on Resol VBus:
I've installed the datalogger DL2 and connect it to my solar controller. All information on sensors, pump speed, flow, ... are now stored in the DL2.
You can connect to the DL2 via different ways:

  • Resol Software Center (free): connect to VBus on IP via the DL2 interface: allow you to configure the controller, view vbus data, and upload schema into DL2
  • DL2 web interface: config the DL2, access to live data (text or graph if schema has been uploaded), export data over 5 years, ...
  • Resol App VBusTouch: to be connected to the DL2 and produce live data schema, D-3 historical data graph and daily heat production

And now we can also have a live graph of the solar system in OpenRemote.
1/ install the DL2
2/ via Resol Software Center, create the schema of your solar installation and add some VBus fields on the schema
3/ upload to DL2. Now, you can have access to the live data in graphical mode via the DL2 web interface. Basically, this is a SVG
4/ in Openremote, add a web element in your panel.xml using the HTTP URL of the live DL2 schema (like "http://IP_address/dl2/live/data.svg"
5/ enjoy
