Posted by pz1 at Apr 01, 2013 14:52

The two binaries that are specific to CPU architecture and ARM are :

1) the JVM – you will need to download and install Oracle Embedded SE for ARM. Check which specific version of ARM your ReadyNAS is using. Download the older Java 6 versions to be on the safe side. The download page is here : Oracle Embedded SE.

2) RXTX compiled for ARM. This is only required if you use any protocols that require serial port communication. The download is available from Oracle here: https://blogs.oracle.com/jtc/entry/serial_port_communication_for_java

Other issues are likely to be specific to ReadyNAS itself.

Posted by juha at Apr 03, 2013 06:42

The ReadyNAS tutorial was written before the release of the v2 version of the NV+ and Duo.
Original version of NV+ and Duo use a specific system on a chip design that do not support Java.
This means that at the time the only systems that supported Java were Intel based, hence the x86 specific reference.

The goal of the first step is just to gain ssh root access to your NAS so you can install a JRE/JDK.
