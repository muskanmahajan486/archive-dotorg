Forums : LIRC-like database for IP and rs232 device control
This page last changed on Jan 15, 2015 by desertdog.

What if there was an opensource database that collected all the device control and feedback commands (rs232, IP, TCP, Telnet, soap, rest, etc)?

Then projects like OpenRemote (or even commercial apps) could rapidly import the needed device codes. The codes would have to be in a standardized format. OpenRemote Beehive could pull in these codes, like it does with the LIRC data.

Maybe this could be added to the LIRC project or a whole new project could be created. Many of the codes are available on remotecentral.com.
