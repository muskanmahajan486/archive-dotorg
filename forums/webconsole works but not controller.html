Hi Mike,

OK I thought I'd check the obvious first and wasn't sure if you were using the android console or the webconsole on your mobile.

If you can access the webconsole but not the controller then it suggests there is something wrong with the controller web app during startup that is causing it to fail. The best place to look for clues will be the console window; look for any error messages that appear when you start OpenRemote and also check through the various log files in the logs directory.
