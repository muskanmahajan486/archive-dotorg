Posted by tobo at Oct 16, 2013 20:36

Another good learning experience, got it working using String format in the end.

rule "Update volume livingroom"
when
$volume:Range (source == "Receiver virtual sensor")
then
execute.command("Receiver volume set", String.format("%03d", (Integer.valueOf($volume.getValue()))));
end
