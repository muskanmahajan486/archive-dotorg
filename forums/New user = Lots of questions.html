Comments:

For X10 with status updates, you will need to look at installing mochad, which is described here: https://sourceforge.net/apps/mediawiki/mochad/index.php?title=OpenRemote

It will require a Linux machine though. The cross-platform implementation for X10 as it currently stands is for sending commands only.

If you're successful with mochad, we'd be interested to know so we can also document the steps on this site.
