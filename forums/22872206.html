It worked for me too, but I got the problem that OR reload the page every ten seconds even though I don't watch that particular page. If you have a number of pages to be reloaded every ten seconds, it generates a lot of traffic.

I solved it with a script that reloads the page and a button that activates the script. It has worked well for me.

I wrote like this:


<script>
function myFunction()
{
location.reload()
}
</script>

</head>

<body>

<div

<input type="button" name="Reload" value="Reload" onclick="myFunction()"style="position:absolute;width:100px;left:42px;top:27px;" >
</div>

</body>

</html>
