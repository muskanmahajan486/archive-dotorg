Did you remember to set the port forwarding?

To enable controller discovery from the emulator, port forwarding must be configured:

  > cd android-sdk-mac_86/tools
  > ./adb forward tcp:2346 tcp:2346

You can see the ADB docs on 'forward' for the exact reasoning why this is required – IIRC the emulator needs to bridge to your actual localhost ports if you run the controller in the same machine, otherwise 'localhost' maps to what the emulator considers its own ports (where controller is not running) instead of the ports of the local machine.
