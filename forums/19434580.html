Comments:

I do not believe there is a complete list of the products directly supported.
For native support, if you look at the protocol that can be used when creating a new command using the on-line modeler, this should give you the complete list.
There is not much A/V device native support in there, either you should use IR, either it can works over a "standard" UDP/TCP/HTTP/Telnet protocol, either you need to implement a new protocol.
For IR, if you do a "Import IR Commands" in the on-line designer, you'll see the full list of currently supported IR codes.
If you have IR codes that needs to be added, you can always post on the forum and we'll see how we can have them added to the database.
Also, if you need support for native protocol for other device, the best is always to post on the forums, so if there is interest from the community, it might encourage somebody to implement it.
