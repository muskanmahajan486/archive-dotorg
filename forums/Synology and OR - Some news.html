Information for all synology owner, a new DSM has been released in Beta 4.2
and the good surprise is that some new official packages are available.
Among those packages, there is a new "Java Manager" package to manage the java installation on your NAS.
There is also a "TOMCAT" package inluding the Tomcat Gui Manager. It's more easy to manage several "java/jsp" application running on the same application server than having each application running its own Tomcat server, like, I think its the case with the current OR Synology package.

Exactly what is needed to deploy the OR on the synology and keep the full flexibility of your "custom" installation.

If you're not aware to Java, Tomcat, ... I'll recommend you to continue with the very good and easy to install synology OR package created by patters on http://pcloadletter.co.uk/2011/10/25/openremote-package-for-synology/
