I'm considering purchasing the following ethernet IO board: http://www.futurlec.com/Ethernet_Controller.shtml

This would be coupled with the following Opto Isolated input board: http://www.futurlec.com/DC_Opto_Input_8.shtml

Looking at the documentation for the board, there appears to be a socket protocol for reading and writing registers.

ie: Simple VB example to print register value from event received from socket on board:

Private Sub Winsock1_DataArrival(ByVal bytesTotal As Long)
   Winsock1.GetData a$, vbString, bytesTotal      ' Get the data from the buffer
   If bytesTotal = 2 And Left$(A$, 1) = "A" Then  ' Confirm that the Data packet is valid
      PortA = Asc(Right$(A$, 1))                  ' Store the value into a register
      Debug.Print Str$(PortA)                     ' Tell the user the value read
   End If
End Sub
