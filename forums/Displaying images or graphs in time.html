Posted by rickcn at Sep 24, 2011 16:33

You could use a web element for this, it will not rendered an image but a web container but this should work (and has been used for this before).

The web element is already supported by the consoles and controller, see this thread : http://www.openremote.org/pages/viewpage.action?pageId=15532976&focusedCommentId=16482448#comment-16482448
Support for it has already been added to the modeler in this branch : https://openremote.svn.sourceforge.net/svnroot/openremote/workspace/selva/UIDesigner_AllSensors_WebView
and will eventually be merged and deployed into the on-line designer.
