The bug faces with "Custom state items" of custom sensor, I get duplicated items any time I press on the submit button.

To reproduce this bug, create a new sensor, set its name, command, type as custom and add some states, then click on submit. From now, any time you edit the sensor and click on submit button, the Designer duplicate its "Custom state items".

In order to check the duplicate items, and since they are not visible in the sersor edit window, you should add a new image in the Designer UI and bind it to the sensor. Now you can see all duplicates.
