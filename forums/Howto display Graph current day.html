Hi Bart,

Start and end time fields are able to accept

1) Constant date in format yyyyMMdd-HH-mm
Example: "20150201-00-00"

2) Variable time derived from current time with optional time offset specified. Prefix '-' or '+' denotes what offset should be used. The offset is specified using "Polling interval string".
Examples:
"-24h" ... current time minus 24 hours
"+4m" ... current time plus 4 minutes
"+5s" ... current time plus 5 seconds
" " ... current time

The quotes are not of course part of the text you specify into the fields. But without the quotes it is hard to show that the End time is a string consisting of one space.

When I wanted to show a graph with values within last 24 hours I used:
Start time: "-24h"
End time: " "
