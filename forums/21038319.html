Ron,

I am using EG. You can see here how I got OR to receive commands from EG:

http://www.openremote.org/pages/viewpage.action?pageId=21037362

As for sending from OR to EG, I am using a shell execute from within OR that calls a batch file that does the actual EG work.
In the OR shell command I define a parameter that I then fetch with the bat file.
The batch looks like this:

@echo off
"C:\Program Files\EventGhost\EventGhost.exe" -n localhost:1024 pass name_of_event %1

where 1024 is the port number of the network event receiver, pass the password, name_of_event the name of the EG event and %1 the parameter received from OR.

Of course you need to add the network event receiver in EG to be able to receive the events.

If you have any doubt...shout!
