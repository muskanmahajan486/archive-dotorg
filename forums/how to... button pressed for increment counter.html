I believe your KNX command to read the dimmer value will be a 1-byte DPT (5.001).

For the custom sensor you can leave the name value pairs empty (this is there to map between values received and values you actually want to be output from the sensor), the problem you have is your read command. Once you have the label updating you should be able to change to use a level sensor and this can be linked to a slider.
