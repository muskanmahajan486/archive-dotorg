Hi Dayve -

I just started up with that myself. What helped for understanding the basic was this documentation about the Wifi-Controller: http://www.limitlessled.com/dev/

In a nutshell:

  • it is important to link the Mi-Light Wifi Controller to your own Wifi (and need to figure the IP within it)
  • the port to use is 8899
  • you need to use the UDP-command and send the mentioned hex commands of the above docu

What I achieved so far was swithing on- and off the lights. So far, i still strugle to sent the right commands for the color selection leveraging the OR color picker. Perhaps you have more success that I have.

I hope that helps.
