Hi all!,

finally I've got the SOLUTION!

For everyone with the same problems, there's a simple solution.

1) Enter into the designer web ( http://composer.openremote.org/demo/login.jsp ) with your user/pass
2) Click into the "UI Designer" and create a "new panel" (don't worry if you are not using it later")
3) Click on "save" and "export as a zip"
4) With your controller launched, go to: ( http://localhost:8080/controller/ ) and enter now your user/pass

Now it should work!!! It works for me.
It seems just like if the Designer need something configured on it in order to initialize the user (It's just a supposition).

Regard!
