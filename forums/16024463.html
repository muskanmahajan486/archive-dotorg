The full REST support (PUT, DELETE, etc) has not yet been completed.

You may want to create an implementation of a new command protocol using the REST for your particular device to control:

Developer How To - Adding Your Own Protocol to OpenRemote Boss 2.0 Controller

There's nothing sufficiently generic to handle all possible REST configs yet.
