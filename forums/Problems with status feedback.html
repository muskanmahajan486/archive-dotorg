Posted by louise35 at Dec 10, 2013 11:15

Somehow your OpenRemote objects don't fit together.
For a slider that controls a dimmer you need:

  • a command that can sent the absolut dimmer value (command 'scale', DPT 5.001) (write command)
  • a command that reads the dimmer value (command 'status', DPT 5.001) (read command)
  • a level sensor linked to the read command
  • a slider that uses the level sensor and the write command

You seem to have linked the read command to the slider, but the slider needs a sensor (which uses the read command) and an extra write command.

Go back to 5.001.
Also, your slider is using the wrong sensor type.
It has to be a switch sensor.
Also your group address 0/0/4 is used for DPT1.001 and DPT5.001 feedback and for write and read.

OpenRemote does not support one group address for action and status.
