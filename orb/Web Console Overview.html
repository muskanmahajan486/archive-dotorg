OpenRemote Box : Web Console Overview
This page last changed on Feb 18, 2009 by juha.

OpenRemote Web console is a generic HTML-based web interface for any web-enabled panel hardware. This includes existing desktop PCs, laptops, web-capable smart phones, touch panels and so on. Any device that has a standards compliant web browser with JavaScript capability can be used as a console device for an OpenRemote system.

Technical Overview

Panel and console web interface is served from OpenRemote Box to connecting clients over HTTP. The web interface is GWT based, therefore a JavaScript capable web browser is a requirement.

The user interface is generated on the ORB based on a XML descriptor provided by UI Composer tools. The descriptor is generic and used to render UI for both web and native console clients. The ORB also provides additional user interface resources, such as icon images, over a specified URL.

Commands will be sent from the web interface as HTTP POST requests. Based on a UI component identifier, the controller will then translate the command to a message that will be routed towards a specific physical port or protocol gateway, such as a KNX command over IP network.
