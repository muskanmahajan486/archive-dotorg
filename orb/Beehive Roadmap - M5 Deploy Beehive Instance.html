Deploy Beehive

Deploying Beehive requires a hosted instance of a MySQL database and either a Tomcat or JBoss application server. These software must be installed and configured properly to serve live clients accessing Beehive data.

Suitable documentation should be provided for administrators to create instances and manage Beehive once it is running.

Deliverables

  1. Administrator documentation on required database, web server, Java middleware installation and configuration to deploy a functional Beehive instance
  2. A live Beehive instance on a public web server
