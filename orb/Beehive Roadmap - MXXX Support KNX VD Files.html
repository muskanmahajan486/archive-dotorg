KNX VD File Support

KNX ETS tool supports visual configuration of KNX-enabled devices. However, there is no auto-discovery of configurable properties of a device. Instead, the metadata is provided in the form of a VD file.

There is currently no centralized repository of such VD files. They must be requested from the KNX device manufacturers. Therefore there's a value in managing the VD metadata as part of Beehive. Further, having a community access and sharing of the files would help with issues such as localization.
