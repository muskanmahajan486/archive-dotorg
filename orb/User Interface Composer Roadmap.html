This document describes the roadmap for UI composer project. It is divided into versions and (hopefully) clear milestones to track the progress.

For an overview of the goals of the project, see User Interface Composer Overview.

Note that milestones do not attempt to divide work into equally sized work units – some milestones may be single feature implementations accomplished in couple of days while others may be larger units of work requiring a week or two to complete.

Note that all the version and milestone information should be considered tentative. This is intended as a live document that can change at any time.
