OpenRemote Box : Infrared Integration Software
This page last changed on Feb 14, 2009 by juha.

This is an overview page to an end-to-end infared solution via OpenRemote Box (ORB). For the hardware, assume a OpenRemote Hardware Reference Implementation equipped with an infrared transmitter tested and compatible with OpenRemote Box.

The infrared software in this iteration is based on LIRC on top of Linux operating system. Therefore a LIRC supported infrared hardware is required. Our hardware pages list infrared transmitters we've tested with ORB hardware and software stacks.

LIRC is invoked by controller software based on a command sent by OpenRemote panel over a HTTP (REST) interface.
