Interoperability and Integration Testing

Beehive exposes REST and JSON APIs for other applications to integrate with its data. In order to ensure that existing application integration remains functional, a test suite specifically targeted for API interoperability should be developed.

As the APIs evolve with new release versions, this test suite must ensure that applications relying on an earlier deployed version of the API continue to function flawlessly.

Deliverables

  1. A test suite that be run from command line build tools which ensures no incompatible changes has been introduced to the REST and JSON APIs as new Beehive versions are released.
