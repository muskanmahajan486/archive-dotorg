Create a web interface

The goal is to allow easier navigation of the remote control configuration files.

Initial interface can be as simple as selecting vendor which exposes available models which allows viewing the LIRC configuration.

It should be possible to retrieve a lircd.conf file via the web interface (in text form that can be deployed for lircd).

Deliverables

JBoss AS deployable WAR with web interface to Beehive IR schema.
