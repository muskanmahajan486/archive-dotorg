Versioning of Data

Both Beehive and LIRC configuration file collection is a live collection of data. Input and contributions from the community is expected on a regular basis.

In order to facilitate and manage community contributions to Beehive, it should be possible to:

  1. add new configuration file entries
  2. mark existing configuration entries as removed
  3. purge existing configuration entries from the database
  4. allow updates of existing configuration entries
