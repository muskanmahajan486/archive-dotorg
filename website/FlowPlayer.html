{FlowPlayer} Macro

FlowPlayer SWF and JavaScript files are stored on this page as attachments.

The flow player macro can be used as follows:

{flowplayer:width=100|height=100}
  [space:page^attachment.mov|swf|flv]
{flowplayer}

For video, H.264 encoding is supported (Flash 9 and above).
