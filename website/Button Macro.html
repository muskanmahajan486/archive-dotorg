{button} Macro

Button macro creates a visual link button with OpenRemote style.

An example:

 {button:title=This Title|description=Some Description|icon-top=5px|icon-left=5px;}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

This Title
Some Description

Available parameters:

  • icon-top
  • icon-left
  • title
  • description
  • twoline
  • label

Icon-Top and Icon-Left

 {button:title=This Title|description=Some Description|icon-top=-15px|icon-left=-15px;}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

This Title
Some Description

Title

 {button:title=Another Title|description=Some Description|icon-top=5px|icon-left=5px;}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

Another Title
Some Description

Description

 {button:title=Another Title|description=With New Description|icon-top=5px|icon-left=5px;}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

Another Title
With New Description

Twoline (not currently implemented)

 {button:title=Another Title|description=Split Long Descriptions <br/> into Two Lines|icon-top=5px|icon-left=5px;|twoline=true}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

Another Title
Split Long Descriptions into Two Lines

Label

 {button:title=Another Title|description=With New Description|icon-top=5px|icon-left=5px;|label=buy}
   !website:OpenRemote Images^iPhoneDownloadButton_72x69.png!, [http://www.openremote.org]
 {button}

Results in:

Another Title
With New Description

Valid attribute values:

  • Buy
