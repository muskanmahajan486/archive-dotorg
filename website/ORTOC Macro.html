{ortoc} Macro

Creates a standard look for table-of-contents sections across the website.

Only level 2 headers (H2) are shown by default. Therefore,

 {ortoc}

yields:

  • Chapter 1
  • Chapter 2
  • Chapter 3

For more depth, depth parameter can be specified:

  {ortoc:depth=4}

which yields:

  • Chapter 1
    • Section 1.1
      • Section 1.1.1
      • Section 1.1.2
      • Section 1.1.3
    • Section 1.2
    • Section 1.3
  • Chapter 2
    • Section 2.1
      • Section 2.1.1
  • Chapter 3
    • Section 3.1
    • Section 3.2
      • Section 3.2.1
    • Section 3.3
      • Section 3.3.1
      • Section 3.3.2
