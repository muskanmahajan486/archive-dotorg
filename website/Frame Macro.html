Frame features are not supported by older web browsers. When not supported, the frame degrades to a regular rectangular border frame with no shadow

Frame macro allows you to decorate content by creating a frame border around it.

An example:

 {frame}
   h5. Title
     - List Item A
     - List Item B
     - List Item C
 {frame}

Results in:
Title
  • List Item A
  • List Item B
  • List Item C

Available parameters:

  • background-color
  • border-width
  • border-color
  • border-radius
  • shadow-color
  • shadow-size
Background Color
{frame:background-color=#B0D9E6}
  {div:style=width: 100px; height: 50px;}
{frame}

Results in:
Border Width
{frame:border-width=3px}
  {div:style=width: 100px; height: 50px;}
{frame}

Results in:
