{listing} Macro

Visual macro for adding consistent labels for images, source code listings, and so on.

An example:

 {listing:caption=Image Caption Goes Here}
   !website:OpenRemote Images^Info_64x60.png!
 {listing}

Results in:

Image Caption Goes Here

Available parameters:

  • caption (mandatory)
  • margin (optional)

Custom Margin

Margin for the caption can be set by using the CSS margin properties (without the 'margin:' prefix).

 {listing:caption=Image Caption Goes Here|margin=-20px 0px 0px 0px;}
   !website:OpenRemote Images^Info_64x60.png!
 {listing}

Results in:

Image Caption Goes Here
