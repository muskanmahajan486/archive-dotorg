{brand-imgtable} Macro

Utility macro for easily creating a table of images (with thumbnail and link to full image). These are used for documenting images used across the website.

An example:

 {table}
   {brand-imgtable:image=website:OpenRemote Logos^OpenRemote-singleline-full-logo_128x20.png|link=OpenRemote-singleline-full-logo_400x62.png}
     Description of the image here
   {brand-imgtable}
 {table}

Results in:

Description of the image here

© OpenRemote Inc. All rights reserved.

Click Image for Full Size

Notable:

  • The linked image must be an attachment in the current page. It should not use the Confluence link attachment '^' notation.
  • Image is only interpreted as a link if the 'link' parameter is present, otherwise it should be a name of the attached file on the current page (without including the usual Confluence '^' link notation)
  • Link is optional parameter
  • If no copyright is specified, automatically assigned to OpenRemote, Inc.

Available parameters:

  • image (mandatory)
  • link (optional)
  • copyright (optional)

No Link, Non-standard copyright

 {table}
   {brand-imgtable:image=Alert_64x69.png|copyright=Dreamstime Royalty Free License}
     Description of the image here
   {brand-imgtable}
 {table}

Results in:

Description of the image here

© Dreamstime Royalty Free License
