Project Information : What is included in OpenRemote 1.0 Milestone 1?
This page last changed on May 03, 2009 by juha.

OpenRemote 1.0 Milestone 1 is a developer release which includes an end-to-end infrared implementation. All the components to run an user interface on an iPhone or iPod Touch and a controller to translate HTTP requests to infrared (LIRC) codes are included.

The included functionality is demonstrated on the Infrared Demo Video and It works! blog.

To deploy OpenRemote 1.0 Milestone 1 you will need a Linux based computer with JavaVM+Tomcat and LIRC installed. We've tested M1 with CommandIR II infrared transmitter. Other LIRC compatible IR transmitters may also work but we can't make any guarantees at this point. The hardware/operating system/infrared combinations we've tested M1 with are ALIX/Voyage/CommandIR (see OpenRemote RI) and B202/Ubuntu/CommandIR.

The Beehive and User Interface Composer components of Milestone 1 have been deployed and are available online at http://composer.openremote.org/demo. This application can be used to generate the controller.xml, iphone.xml and lircd.conf files necessary for controller deployment.

What is not included

OpenRemote 1.0 Milestone 1 does not include any KNX or X10 functionality. These are scheduled for OpenRemote 1.0 release and will appear in subsequent milestone releases.

The iPhone user interface is limited to the most basic button layout in M1. OpenRemote 1.0 Milestone 2 will extend this functionality significantly allowing more customizable user interfaces.
